#####  Keen Keeper  ####

A modern web application to manage and track interactions with your friends. Stay connected and visualize your social activities in one place!



#### Technologies Ued To Build--->>>
 @ Next.js (Framework)
 @ React   (Library)
 @ Tailwind CSS (Styling)
 @ Recharts (Data Visualization)
 @ React Toastify (Notifications)
 @ 



##  Key Features

1.  Real-time Interaction Logging:  Use to record calls, texts, and video meetings with friends. Each interaction
    is instantly saved and displayed on your timeline.
2.  Dynamic Timeline Filtering: Easily filter your interaction history by type (Call, Text, or Video) to find
    specific  memories or logs quickly.
3.  Visual Statistics: A beautiful, interactive Pie Chart that provides a breakdown of your communication habits,  
    helping you see how you spend time with your friends.
